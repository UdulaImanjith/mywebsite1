# **App Name**: PixelCanvas Studio

## Core Features:

- Dynamic Portfolio Showcase: Categorized display of Udula's projects (Graphic, Web, App Dev) with support for image galleries and embedded videos. Project tiles are clickable to view detailed information.
- Interactive User Interface: Modern and clean dark UI with engaging mouse hover effects and distinct button animations for a polished user experience across all sections (Home, About Me, Services, Projects, Testimonials, Contact).
- Comprehensive About Section: Dedicated section detailing Udula's work experience, specialized skills (graphic design, web development, video editing, animation, 3D modeling), and professional approach.
- Client Testimonials & Services Display: Engaging sections to present client testimonials and a clear overview of services offered, reinforcing credibility and expertise.
- AI-Powered Content Refinement Tool: An administrative tool that helps Udula refine or generate concise, impactful descriptions for portfolio projects and services, ensuring consistent messaging and compelling storytelling.
- Admin Content Management: A secure, user-friendly admin interface for Udula to easily add, edit, and categorize portfolio projects (uploading images/videos), manage service details, and update testimonials without needing to write code.
- Integrated Contact Form: A functional contact form allowing potential clients to easily send inquiries, with form submissions securely captured and delivered to Udula.

## Style Guidelines:

- Background color: Deep black (#0D0D0D) to establish a sleek, modern, and dark aesthetic, providing high contrast for other elements.
- Primary color: Vibrant dark orange (#E07219). This will be used for key headings, call-to-action buttons, and interactive elements to provide energy and focus within the dark theme.
- Accent color: Muted golden yellow (#F0D77D). Used sparingly for subtle highlights, interactive states, and small decorative details to add a touch of sophistication and warmth, complementing the primary orange.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-savvy, and clean look, suitable for conveying a sense of innovation and precision.
- Body font: 'Inter' (sans-serif) for clear readability and a neutral, professional feel, ideal for project descriptions, 'About Me' details, and other body text.
- Utilize relevant and professionally designed line-art or minimalist icons to visually represent different sections, services, and project categories, enhancing clarity and visual appeal without clutter.
- Implement meticulous control over margins and paddings to create a clean, airy, and well-structured layout, ensuring visual balance and an optimal reading experience, especially within the dark UI.
- Subtle, fluid mouse interaction effects (e.g., hover states, scaling, or glowing) and elegant button animations to enhance interactivity and provide positive user feedback throughout the portfolio, reinforcing the modern design.
